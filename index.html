Task:

  1. The initial positioning of the tokens is such that the first two columns have green tokens and the remaining columns have red tokens;
  2. For each token, a number between 1 and 4 is drawn;
  3. One by one, each token is swapped places with a token located in the same row, in the column with the number drawn in point b.
  4. After all the tokens have been swapped, operations b-c are repeated four more times.

Modified: the board can be of size MxN (limited to 25 each, because why many?) and the number of iterations can be from 5 to 10 (limited for the same reason).
